# Theming & Skins

Use this section to customize MamoPlayer visuals, layouts, and branding behavior.

Theme and layout configuration lives on `ProMamoPlayer` via `themeName`, `theme`, `layoutVariant`, and `icons`.

- Built-in themes: `light`, `dark`, `ott`
- Layout variants: `compact`, `standard`, `ott`

See [Pro Player](../pro-player/) for full prop examples.
