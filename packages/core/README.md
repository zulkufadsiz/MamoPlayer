# @mamoplayer/core

Core React Native video player package for the MamoPlayer SDK.

## Installation

```sh
npm install @mamoplayer/core react-native-video
# or
yarn add @mamoplayer/core react-native-video
```

## Peer Dependencies

| Package | Version |
|---------|---------|
| `react` | `>=18` |
| `react-native` | `>=0.72` |
| `react-native-video` | `>=6` |

## Usage

```tsx
import MamoPlayer from '@mamoplayer/core';

<MamoPlayer
  source={{ uri: 'https://example.com/video.mp4' }}
/>
```
